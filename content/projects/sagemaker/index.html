sagemaker R Interface

sagemaker-tidymodels: framework for tidymodels

The sagemaker R package provides a simplified interface to the AWS Sagemaker API. See the documentation to get started.

sagemaker-tidymodels is a Python package and Docker container that creates an AWS Sagemaker framework for training and deploying machine learning models created by tidymodels in the cloud.
